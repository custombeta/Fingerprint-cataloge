# Changelog

## [1.0.1] - 2025-03-06

### ✨ Новое
- Добавлен секретный отпечаток, включается при определенном действии...
- Добавлена поддержка анимированных отпечатков (формат `.fp`)
- Новый экран создания отпечатка с превью иконки
- Автоматическая проверка обновлений при запуске
- Поддержка тем: светлая, тёмная, системная
- Импорт отпечатков из архивов `.fp`
- Кнопка перезапуска SystemUI без перезагрузки устройства
- Поддержка Monet (Material You) цветовой схемы
- Xposed модуль для замены системного отпечатка

### 🔧 Улучшения
- Оптимизирована загрузка иконок через Coil
- Ускорен старт приложения
- Улучшена анимация переходов между экранами
- Переработан интерфейс настроек
- Снижено потребление RAM при работе в фоне
- Улучшена стабильность Xposed хука

### 🐛 Исправления
- Исправлен краш при импорте повреждённого архива
- Исправлено дублирование отпечатков при повторном импорте
- Исправлена утечка памяти в FingerprintManager
- Исправлено некорректное отображение на устройствах с вырезом
- Исправлен баг с нечитаемым текстом в тёмной теме

### ⚠️ Известные проблемы
- На некоторых устройствах MediaTek возможен высокий SUnreclaim
- Xposed хук может не применяться без перезагрузки после первой установки
